#Chocolate Sales Analysis Project

## Overview
This project analyzes chocolate sales data using Python to extract business insights such as top products, country-wise sales, and salesperson performance.

---

## Tools Used
- Python
- Pandas
- Matplotlib

---

## Key Features
- Data Cleaning (Currency conversion)
- Sales analysis by Country
- Sales analysis by Product
- Salesperson performance analysis
- Bonus decision system
- Top product analysis
- Total sales 

---

## Visualizations
- Bar chart: Country sales
- Horizontal bar chart: Product sales
- Bar chart: Salesperson performance
- Dashboard view (subplot layout)

---

## Business Logic
- Salesperson gets 20% bonus if sales > 50,000
- Otherwise no bonus is given

---

## Insights
- Dark chocolate is a top-selling product
- Few countries dominate revenue
- Sales distribution is uneven across employees
